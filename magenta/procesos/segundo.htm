INSTRUCTIVO GENERAL DEL PROGRAMA AFINIDAD

 

 

Cómo actualizar los datos de un Usuario AAA o BBB:

 

 

 

 

 

1. Una vez en la página de información del Usuario, donde se sabe que tipo de Usuario es, damos clic en el botón actualizar

 

2. El sistema nos solicita el Usuario y la clave (generalmente son el código de nómina y los últimos cuatro números de la cédula)

 

3. El sistema nos presenta el formulario para actualizar los datos del cliente

 

4. Tras diligenciar el  formulario, activamos el botón DATOS COMPLETOS Y damos clic en el botón actualizar

 

4. Aparece un  mensaje sobre el éxito de la operación con nuestro nombre

 

5. Para ingresar otros familiares de un Usuario AAA, diligenciamos la información del familiar y actualizamos la información. El

proceso se repite por cada familiar que deseemos ingresar.
