INSTRUCTIVO PARA MAGENTA Y COMUNICACIONES

                                                                                            DEL PROGRAMA: AFINIDAD

 

 

Cómo clasificar manualmente o actualizar los datos de un Usuario:

 

 

Elija en el menú de magenta la opción:  AAA

 

 

1. Ingrese el documento de identidad y el tipo de documento en la página de bienvenida

 

 

2. Una vez en la página de información del Usuario, donde se sabe que tipo de Usuario es, damos clic en el botón actualizar

 

 

3. El sistema nos solicita el Usuario y la clave (generalmente son el código de nómina y los últimos cuatro números de la cédula)

 

4. El sistema nos presenta el formulario para actualizar los datos del cliente, en el podemos clasificar al Usuario con las siguientes opciones:

  • AFIN AAA: Usuario del programa Afinidad, actualizable por cualquier empleado de la Clínica

  • AFIN BBB: Familiar de un Usuario AAA,  actualizable `por cualquier empleado de la Clínica

  • VIP AAA: Usuario del programa Afinidad, actualizable solo por comunicaciones

  • VIP BBB: Familiar de VIP AAA, actualizable solo por comunicaciones

  • NO CLASIFICADO: Fuera de la base de datos del programa Afinidad

  • DESCLASIFICADO:  Aparece en la base de datos del programa de afinidad, para que no sea clasificado como AAA

 

5. Tras diligenciar el  formulario, activamos el botón DATOS COMPLETOS Y damos clic en el botón actualizar

 

4. Aparece un  mensaje sobre el éxito de la operación con nuestro nombre

 

6. Para ingresar otros familiares de un Usuario AAA, diligenciamos la información del familiar y actualizamos la información. El

proceso se repite por cada familiar que deseemos ingresar.
